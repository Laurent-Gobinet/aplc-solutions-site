# APLC Solutions - Starter Next.js V3 Premium Photos

Cette version est prête pour GitHub/Vercel.

Les photos premium sont chargées en ligne via Unsplash Source pour valider le rendu sur Vercel. Pour la version définitive, il faudra remplacer ces URLs par des images téléchargées/licenciées dans `/public`.

## Lancement
npm install
npm run dev

## Déploiement
Importer le repo GitHub dans Vercel, puis connecter `aplc-solutions.fr`.
